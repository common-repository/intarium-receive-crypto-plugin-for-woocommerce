This plugin integrated in WooCommerce allows merchants to be paid in Bitcoin or DASH, while the customer pays directly in Euro, USD or CHF.

== Description ==
Add your free API Token after registration from Intarium and activate the payment method. More information can be found at https://intarium.ch

== Installation ==
So that this payment method is ready for use, all you need after the installation and activation of the plugin is to store your individual access authorization (API token) under WooCommerce -> Settings > Payments -> "Intarium Receive Crypto" in WordPress.
You can find your API token after complete registration on Intarium.ch in the API settings.
